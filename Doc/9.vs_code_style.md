# SnmpNms GUI를 VS Code 스타일로 변경하는 계획

## 개요

SnmpNms 애플리케이션의 GUI를 VS Code 스타일로 변경하여 현대적이고 일관된 사용자 경험을 제공합니다.

---

## 현재 구조 분석

### 현재 레이아웃
- **상단**: Menu Bar + ToolBar (2개)
- **좌측**: Selection Tool (GroupBox, TabControl)
- **중앙**: View Window Area (TabControl)
- **하단**: StatusBar + Event Log

### 현재 컴포넌트
- `MainWindow.xaml`: 메인 창 레이아웃
- `Selection Tool`: Map Tree, Device List, Trend, Event, Menu 탭
- `View Window Area`: Map View, Device Details, MIB Tree, MIB Table, MIB Graph, SNMP Test, Event Log 탭
- `Event Log`: 하단 고정 영역

---

## VS Code 스타일 특징

### 레이아웃 구조
1. **Activity Bar** (왼쪽 가장자리)
   - 세로 아이콘 버튼
   - 현재 활성화된 뷰 표시
   - 클릭 시 Sidebar 내용 변경

2. **Sidebar** (왼쪽 패널)
   - Activity Bar 선택에 따라 내용 변경
   - 접기/펼치기 가능
   - 트리 뷰 스타일

3. **Editor Area** (중앙)
   - 탭 기반 인터페이스
   - 여러 파일/뷰 동시 열기
   - VS Code 스타일 탭 디자인

4. **Panel** (하단)
   - 터미널, 문제, 출력 등
   - 접기/펼치기 가능
   - 여러 패널 탭 지원

5. **Status Bar** (최하단)
   - 상태 정보 표시
   - VS Code 스타일 디자인

## 변경 계획

### 1단계: 레이아웃 구조 변경

#### 1.1 Activity Bar 추가
- **위치**: 왼쪽 가장자리 (세로)
- **너비**: 48px
- **아이콘**:
  - Map (탐색기) - 현재 Selection Tool
  - Search (검색) - 검색 기능
  - Event Log (문제) - 이벤트 로그
  - Settings (설정) - 설정
- **기능**: 클릭 시 Sidebar 내용 변경

#### 1.2 Sidebar로 전환
- **위치**: Activity Bar 오른쪽
- **너비**: 280px (조절 가능)
- **내용**:
  - **Map**: 현재 Selection Tool (Map Tree, Device List 등)
  - **Search**: 검색 기능 (향후 구현)
  - **Event Log**: 이벤트 필터 및 목록
  - **Settings**: 설정 패널 (향후 구현)
- **기능**: 접기/펼치기 가능

#### 1.3 Editor Area 유지

- **기능**: 현재 TabControl 유지
- **스타일**: VS Code 스타일 탭 디자인 적용

#### 1.4 Panel 추가
- **위치**: 하단 (StatusBar 위)
- **높이**: 220px (조절 가능, 최소 120px)

#### 1.5 Status Bar 유지

## 구체적 변경 사항

### MainWindow.xaml 구조 변경

#### 현재 구조
```xml
<DockPanel>
  <Menu DockPanel.Dock="Top"/>
  <ToolBarTray DockPanel.Dock="Top"/>
  <StatusBar DockPanel.Dock="Bottom"/>
  <Grid>
    <Grid.ColumnDefinitions>
      <ColumnDefinition Width="280"/>  <!-- Selection Tool -->
      <ColumnDefinition Width="5"/>   <!-- Splitter -->
      <ColumnDefinition Width="*"/>   <!-- View Window Area -->
    </Grid.ColumnDefinitions>
    <Grid.RowDefinitions>
      <RowDefinition Height="*"/>
      <RowDefinition Height="5"/>
      <RowDefinition Height="220"/>   <!-- Event Log -->
    </Grid.RowDefinitions>
  </Grid>
</DockPanel>
```

#### 변경 후 구조
```xml
<DockPanel>
  <!-- Activity Bar (왼쪽 가장자리) -->
  <ActivityBar DockPanel.Dock="Left" Width="48"/>
  
  <!-- Sidebar (왼쪽) -->
  <Sidebar DockPanel.Dock="Left" Width="280"/>
  
  <!-- Status Bar (하단) -->
  <StatusBar DockPanel.Dock="Bottom"/>
  
  <!-- Panel (하단, StatusBar 위) -->
  <Panel DockPanel.Dock="Bottom" Height="220"/>
  
  <!-- Editor Area (중앙) -->
  <EditorArea/>
</DockPanel>
```
